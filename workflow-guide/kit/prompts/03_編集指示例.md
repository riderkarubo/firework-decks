# 03 編集指示例（コピペ集）

よく使う編集指示をそのままCoworkに貼れる形でまとめています。
**【】の部分を実際の番号・セクション名に書き換えてから使ってください。**

> ※ 注意：カードを追加・削除・統合・並び替えした後は、**必ず通し番号の振り直しを行ってください**。振り直しを忘れると No.1 の次が No.3 になるなど番号が飛びます。

---

## カード削除

```
No.【X】を削除してください。削除後、残ったカードの通し番号を No.1 から順に振り直してください。
```

---

## カード統合

```
No.【X】と No.【Y】を統合してください。内容は【X】をベースに【Y】のポイントを加筆する形でまとめてください。統合後、通し番号を No.1 から順に振り直してください。
```

---

## セクション削除

```
「【セクション名】」のセクション全体を削除してください。削除後、残ったカードの通し番号を No.1 から順に振り直してください。
```

---

## セクション並び替え

```
セクションの順番を以下の順に変えてください：
1. 【セクション名A】
2. 【セクション名B】
3. 【セクション名C】

変更後、カードの通し番号を上から順に No.1 から振り直してください。
```

---

## 文章修正

```
No.【X】の「idea-text」（本文）を以下の内容に書き換えてください：

【新しい本文をここに書く】
```

---

## カード追加

```
「【セクション名】」セクションの末尾に新しいカードを追加してください。

タイトル：【企画タイトル】
本文：【企画内容】
イチオシ：【はい / いいえ】

追加後、通し番号を No.1 から順に振り直してください。
```

---

## 通し番号の振り直し（単独）

```
全カードの通し番号を、上から順に No.1、No.2、No.3 … と振り直してください。セクションをまたいでも連続番号にしてください。
```

---

## イチオシ（★）の付け外し

```
No.【X】にイチオシマーク（★ イチオシ）を追加してください。左帯の色もマゼンタ（#fa006d）にしてください。
```

```
No.【X】からイチオシマーク（★ イチオシ）を外してください。左帯の色をデフォルト（グレー）に戻してください。
```

---

## 根拠（データ）の追加

```
No.【X】に根拠データを追加してください。

根拠テキスト：【ここにデータ・出典を書く】
```
