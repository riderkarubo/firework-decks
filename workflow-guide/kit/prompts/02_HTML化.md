# 02 HTML化プロンプト

上で出した企画案を、以下の仕様でHTMLにしてください。

---

## デザイン仕様

- スタイル：マガジン風・読み物系。白背景に黒テキスト、アクセントカラー `#fa006d`（マゼンタ）
- フォント：Noto Sans JP（Google Fonts）
- カード形式：各企画案を「idea-strip」カードで表示
- 通し番号：上から順に No.1、No.2、No.3 … と必ず振る（セクションをまたいでも連続番号）
- レスポンシブ：スマホでも崩れないこと
- 印刷対応：`@media print` で背景色を維持

## 構成

1. **カバー**：タイトル・案件名eyebrow・ターゲット/KGI/出演者/最終更新の4統計バー
2. **設計思想ブロック**：企画全体の方針を1文blockquoteで。その下にフロー矢印（視聴→…→KGI）
3. **出演者の役割**：cast-cardで各出演者の役割を簡潔に
4. **セクション別カード（SEC 01〜）**：各セクションに見出し＋企画カードを並べる。イチオシ案は `class="pick"` を付けて左帯をマゼンタにする
5. **フッター**：「クライアント名様 / Last Updated: 日時」

## 注意事項

- `idea-id` の番号はセクションをまたいでも連続番号（No.1, No.2, No.3 …）にしてください
- `gate.js` のscriptタグは含めないでください。後で追加します
- `<meta name="robots" content="noindex, nofollow">` は含めてください
- `<meta name="last-modified" content="...">` を含めてください（日時は今日の日付で）
- 本文中に「税抜」「税込」は書かないでください
